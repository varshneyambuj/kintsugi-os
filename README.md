# Kintsugi OS

An operating system for sub-1 watt computers.

Kintsugi OS is a hard fork of [HaikuOS](https://www.haiku-os.org), targeting low-power edge devices and agent-native computing. The BeOS/Haiku kernel and kit architecture, stripped and reshaped for milliwatt-class hardware.
