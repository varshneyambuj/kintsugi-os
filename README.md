# Kintsugi OS

An operating system for sub-1 watt computers.

## Acknowledgements

Kintsugi OS builds upon the work of the [Haiku](https://www.haiku-os.org/) project. We are grateful to the Haiku developers for creating and maintaining an open-source operating system that has served as the foundation for this project.

We also thank the team behind [BeOS](https://en.wikipedia.org/wiki/BeOS), whose innovative operating system inspired the Haiku project.
